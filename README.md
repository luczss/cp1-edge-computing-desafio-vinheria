Descrição do Projeto
Este projeto foi desenvolvido para a Vinheria Agnello com o objetivo de monitorar as condições ambientais do local onde os vinhos são armazenados.

A qualidade do vinho pode ser afetada por fatores como temperatura, umidade e luminosidade. Pensando nisso, criamos um sistema inteligente capaz de acompanhar essas variáveis em tempo real e alertar quando os valores estiverem fora do ideal.

Objetivo:

-Garantir melhores condições de armazenamento dos vinhos, ajudando a preservar sua qualidade e aumentando a eficiência do controle do ambiente.

Funcionalidades:
-Monitoramento de temperatura
-Monitoramento de umidade
-Monitoramento de luminosidade
-Alertas visuais quando valores estiverem inadequados
-Simulação no Tinkercad

Tecnologias Utilizadas:

Como Executar:

Acesse o projeto no Tinkercad
Clique em Iniciar Simulação
Observe os sensores e os alertas do sistema
Problema Resolvido
Antes do sistema, o controle era manual e sujeito a falhas. Agora, o monitoramento ocorre automaticamente, trazendo mais segurança e praticidade.

Conclusão:
O projeto oferece uma solução simples, eficiente e acessível para monitorar a luminosidade do ambiente, auxiliando a Vinheria Agnello a preservar a qualidade dos seus produtos. Com o uso de sensores e alertas visuais e sonoros, o sistema permite uma resposta rápida a variações de luz, contribuindo para melhores condições de armazenamento e maior confiabilidade no controle do ambiente.
