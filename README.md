## 📌 Descrição do Projeto

Este projeto foi desenvolvido para a Vinheria Agnello com o objetivo de monitorar as condições ambientais do local onde os vinhos são armazenados.

A qualidade do vinho pode ser afetada por fatores como temperatura, umidade e luminosidade. Pensando nisso, foi criado um sistema automatizado capaz de acompanhar essas variáveis em tempo real e emitir alertas quando os valores estiverem fora do ideal.

---

## 🎯 Objetivo

Garantir melhores condições de armazenamento dos vinhos, ajudando a preservar sua qualidade e aumentando a eficiência no controle do ambiente.

---

## ⚙️ Funcionalidades

* Monitoramento de temperatura
* Monitoramento de umidade
* Monitoramento de luminosidade
* Alertas visuais e sonoros
* Simulação no Tinkercad

---

## 🛠️ Tecnologias Utilizadas

* Arduino Uno
* Tinkercad
* Sensores virtuais
* Linguagem C/C++ (Arduino IDE)

---

## 🔌 Componentes Utilizados

* Arduino Uno
* Sensor de temperatura
* Sensor de luminosidade (LDR)
* Sensor de umidade (opcional)
* LEDs
* Resistores
* Protoboard
* Cabos jumper

---

## ▶️ Como Executar

1. Acesse o projeto no Tinkercad
2. Clique em **"Iniciar Simulação"**
3. Observe os valores dos sensores
4. Verifique os alertas visuais e sonoros conforme as condições do ambiente

---

## ❗ Problema Resolvido

Antes do sistema, o controle das condições ambientais era manual e sujeito a falhas.
Com a solução proposta, o monitoramento passa a ser automático, trazendo mais segurança, precisão e praticidade.

## ✅ Conclusão

O projeto oferece uma solução simples, eficiente e acessível para monitorar temperatura, umidade e luminosidade do ambiente, auxiliando a Vinheria Agnello a preservar a qualidade dos seus produtos.
Com o uso de sensores e alertas visuais e sonoros, o sistema permite uma resposta rápida a variações ambientais, garantindo melhores condições de armazenamento e maior confiabilidade no controle do ambiente.
